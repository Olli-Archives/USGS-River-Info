Add a River By USGS Site ID

How does it all work??? Great question! This site is capable of displaying parameters such as flow, water temperature, river height etc for all of your favorite rivers (as long as there is a USGS site ID available), and the best part is if you have an account setup with us, we will remember all of yor favorite rivers. Now you can just go to the favorites tab and chose which rivers you are interested on a particular day without needing to search for the USGS ID number ever AGAIN!.

If you are not familiar with USGS site IDs, they can all be found here USGS. While the USGS site is not the easiest to navigate I have allways had better luck search for the USGS site IDs using google. For example if you are looking for the site ID for the Dechuttes river, you could google the following "Dechuttes USGS site ID".

Once you are displaying all the rivers that you want to look at, you can toggle over from the side bar to the "PARAMS" and chose what parameters you want to look at. If you have a river that you know you will like to get information from in the future, you can add the river to your favorites by clicking the star symbol. Now you can just go over to the "FAVORITES" on the side bar and chose which rivers you want to see displayed.
